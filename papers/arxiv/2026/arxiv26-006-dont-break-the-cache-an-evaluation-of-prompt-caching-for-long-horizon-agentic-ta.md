# Don't Break the Cache: An Evaluation of Prompt Caching for Long-Horizon Agentic Tasks

## 摘要

Long-horizon LLM agents accumulate large contexts across many tool calls, but the benefits of prompt caching for such workloads remain underexplored. This work evaluates prompt caching across major providers and compares multiple strategies including full-context caching, system-prompt-only caching, and caching that excludes dynamic tool results. The results show that caching strategy choice matters materially for agentic workloads and that context structure strongly affects achievable cost and latency savings.
