# Cost-Efficient Serving of LLM Agents via Test-Time Plan Caching

## 摘要

Agentic applications incur substantial latency and cost from repeated planning and reasoning. Existing caching techniques designed for chatbots are insufficient when outputs depend on external state and environment context. This work proposes agentic plan caching, which extracts, stores, matches, and adapts structured plan templates from completed executions at test time. Across several real-world agentic applications, plan caching reduces serving cost materially while maintaining task performance.
