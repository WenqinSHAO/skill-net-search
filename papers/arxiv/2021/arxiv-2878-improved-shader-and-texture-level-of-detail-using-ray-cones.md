# Improved Shader and Texture Level of Detail Using Ray Cones

## 摘要

In real-time ray tracing, texture filtering is an important technique to increase image quality. Current games, such as&nbsp;Minecraft with RTX&nbsp;on Windows 10, use ray cones to determine texture-filtering footprints. In this paper, we present several improvements to the ray-cones algorithm that improve image quality and performance and make it easier to adopt in game engines. We show that the total time per frame can decrease by around 10% in a GPU-based path tracer, and we provide a public-domain implementation.
