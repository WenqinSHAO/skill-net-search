# Opportunities for RTL and Gate Level Simulation using GPUs

## 摘要

This paper summarizes the opportunities in accelerating simulation on parallel processing hardware platforms such as GPUs.&nbsp;&nbsp;First, we give a summary of prior art. Then, we propose the idea&nbsp;that coding frameworks usually used for popular machine learning&nbsp;(ML) topics, such as PyTorch/DGL.ai, can also be used for exploring&nbsp;simulation purposes. We demo a crude oblivious two-value cycle&nbsp;gate-level simulator using the higher level ML framework APIs that&nbsp;exhibits &gt;20X speedup, despite its simplistic construction. Next,&nbsp; we summarize recent advances in GPU features that may provide additional opportunities to further state-of-the-art results. Finally,&nbsp;we conclude and touch upon some potential areas for furthering&nbsp;research into the topic of GPU accelerated simulation.
